美国旅游面签后的一些感受

发表于 2017-03-07   |   分类于 游记   |     |   阅读次数

为什么去美国

其实不是自己个人的旅游计划，而是公司的福利，每年奖励优秀员工出国游，但是首先要过的一关就是签证，因为在大家的印象中，美国面签很难过，准确一点就是很容易挂。今天去面签了，谈一谈前后的感受吧

签证的流程

美国面签的流程就不多介绍了，大家可以自己去查查，不过如果大家准备去美国，一定要提前一点去申请面签，一般我觉得最好提前三个月，就是距离你出发的时间，这是针对第一次去美国的，去过的或者其他情况我待会会讲到，因为有一些情况是通过率比较高的。申请面签如果是个人，直接去他们的网站申请，如果是报的团，旅行社会帮你申请，你提供信息就可以了。不过，都必须去领事馆面签。网站上申请的时候会填写的资料，一定要如实填写，然后牢记（待会会说为什么）！！！

申请后进入面签流程

在网申以后，一般会根据那段时间的面签排队人数来定你具体面签的时间，人多可能就要多等，人少可能很快就能排到你了。我们是2月25号约的，然后是3月7号面签。面签时，一般会要要求带一些资料，比如我们是报团，旅行社直接要求的资料有：

然后带上资料，排队，过安检、录指纹、面签

先说说资料

面试官会看准备资料吗？这个问题没有确切的答案，不过我们今天23个人面试，都没有看任何准备的资料，但是大家还是应该将能够准备的资料都带上，而且都要的都是原件。

资金和资产证明

大家其实最关心的应该是资金证明，我个人认为，你能不能签过和你有多少存款，关系真不太大，有存款、固定产其实中心思想还是为了证明你在国内是有牵挂的，你不会去了就不回来了。

空护照

另外很重要的一点就是是否是空护照，就是没有任何出国记录的，从我们公司30多个人（还有其他地区分公司）的面签结果来看，这一点确实会有影响，比如公司有一哥们，空护照，只问了他两个问题：叫什么？什么工作？然后就被拒了！当然不止这一个空护照被拒的，30多个人中有10个人被拒，其中有6个人是空护照，当然被拒的原因应该不止空护照这一个原因。如果你护照有欧洲、澳洲的出国记录（美国就不说了，只要过了一次，以后就很好过了），或者你去了国外很多地方或很多次，一般通过率还是非常高的。

再说说问题

一般的问题按照问的概率来排：你去美国干什么？你是什么工作？结婚了吗？为什么没有带家人等等……我总结了一下，一般就是问你的基本情况，而这些基本情况就是你申请时所填的信息，回答时一定要和之前的填写的能对上，如果对不上，那怕是一点点都过不了。然后出了基本新信息的问题，还会有一些很突然的问题，比如我们公司的HR被问到：

面试官：你是什么工作？
HR：HR
面试官：负责HR哪方面的工作呢？
HR：招聘
面试官：你在工作中面临的最大问题是什么？
HR：快速的识人辨人（反应很快，我就在她旁边）

所以我总结，两类问题1）你的信息；2）你是否因为有其他想法而紧张。所以在面签时，大家一定要落落大方、自信、回答问题简洁和坚定，不要做过多的解释，更不要紧张。

通过率较高和较低的情况

公司的同事也有自费带家属的，如果是一家人去面签，比较好通过，一家人有小孩一起去的比较好过。有一个同事之前在华为和腾讯工作过，旅行社提交他的资料时，将他的这两段经历都删除了，而且嘱咐他面签时千万不要说之前在这两家公司工作过，我也不知道为什么，肯定是有原因的吧。大龄单身好像也不怎么好过

最后总结

其实，美国面签的通过率还是挺高的，但是有偶然性，没有说你怎样了或者你是怎样的就一定过（针对第一次，本文都针对第一次）。因为偶然性太大，所以一定要提前，像这次，由于成都的今天面签，31号出团，所以没过的同事就来不及预约第二次面签了。然后深圳有一个同事，是2月27在广州面的，没有通过，马上又预约了一次也是在今天通过了（感觉成都人很多，因为整个西南加西北的省份中，就成都有美领馆）。
